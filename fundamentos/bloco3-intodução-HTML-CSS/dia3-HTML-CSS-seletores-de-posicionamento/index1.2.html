Lista dos Melhores Sites que já vi

Top 3 dos melhores sites que existem na web

  1. W3Schools
  2. Poly.Land
  3. Arquivo Marxista da Internet

Principais motivos para eu gostar deles

  • O W3Schools é um otimo site gringo para aprender programação, intuitivo e didático.
  • Poly.Land é um dos poucos sites que eu conheço voltado para a questão da não-monogamia, tendo diversos artigos que abordam varios angulos da questão.
  • O marxist.org, apesar de feio e mal diagramado, acaba sendo o maior arquivo de textos marxistas online do mundo, estando disponivel em diversas linguas e sendo constantemente atualizado com novos artigos e traduções.
